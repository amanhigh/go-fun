# Requirements: Go Fun / Kohan

**Defined:** 2026-04-16
**Core Value:** Make journal review fast and reliable from the detail page.

## v1 Requirements

### Journal Review

- [ ] **JDET-01**: User can mark a journal as reviewed today from the detail page.

### Journal Tags

- [ ] **JDET-02**: User can add one or more tags to the journal from the detail page.

### Journal Notes

- [ ] **JDET-03**: User can add a note to the journal from the detail page.

### Save Behavior

- [ ] **JDET-04**: After each edit action, the page reloads to show the latest saved state.

## v2 Requirements

- **JDET-05**: Edit existing tags and notes inline — deferred until the add flows are complete.
- **JDET-06**: Manage journals in bulk — out of scope for this milestone.

## Out of Scope

| Feature | Reason |
|---------|--------|
| Bulk journal editing | The milestone is focused on one journal detail page |
| Tag administration screens | Tagging should stay in-place on the detail page |
| Rich text note editor | Plain text notes are sufficient for this milestone |
| Search and filtering on the detail page | Not required to complete the edit flows |
